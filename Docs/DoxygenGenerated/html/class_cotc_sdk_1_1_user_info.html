CotcSdk.UserInfo Class Reference
Model classes (server/client communication)

Description

Info about a user. Can be enriched with information, accessible using the index operator []. Typically contains a profile field, with displayname, email and lang. You can fetch this by doing string name = UserInfo["profile"]["displayname"];
