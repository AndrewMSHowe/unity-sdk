CotcSdk.GamerOutline Class Reference
Model classes (server/client communication)

Description

Outline information about a player. Can be enriched with information, accessible using the index operator []. Typically contains a profile field, with displayname, email and lang. You can fetch this by doing string name = GamerOutline["profile"]["displayname"];
